分析Dockerfile。发现golang是三个服务。

这里proxy和web服务都是web用户起来的。server是root起的。

看web源码

能命令注入。但是有waf

func waf(c string) bool {
	var t int32
	t = 0
	blacklist := []string{".", "*", "?"}
	for _, s := range c {
		for _, b := range blacklist {
			if b == string(s) {
				return false
			}
		}
		if unicode.IsLetter(s) {
			if t == s {
				continue
			}
			if t == 0 {
				t = s
			} else {
				return false
			}
		}
	}

	return true
}

意思就是。不能带.*?和字母
用https://guokeya.github.io/post/omqnCRpCQ/
