JSON注入

2020-01-02

介绍
JSON是javascript对象表示法

格式如下：

{"name":"123","pass":"456"}
一个名称用冒号对应一个值，双引号包裹，以逗号分割
{"name":["one":"admin","two":"test"]}
还可以包含数组

环境
PHP代码如下：

输入：
json={"name":"admin","passwd"="test"}
<?php
if(isset($_POST['json'])){
	$json=$_POST['json'];
	$json=json_decode($json);
#将JSON解码后赋值
}
if(!$json){
	die('json error');
}
@$conn=mysql_connect('localhost','root','cnm13579.');
mysql_select_db('test');
$username=$json->name;
#输出结果为admin
$password=$json->passwd;
#输出结果为test
$password=md5($password);
$sql="select id,name,passwd from user where name='$username' and passwd='$password'";
#带入sql语句查询，造成JSON注入
echo $sql."<br />";
$query=mysql_query($sql,$conn);
if(@$result=mysql_fetch_assoc($query)){
	echo "id:".$result['id']."<br />";
	echo "name:".$result['name']."<br />";
	echo "passwd:".$result['passwd']."<br />";
}else{
	die(mysql_error());
}
?>

这里同sql注入的方式就可以了，--空格，可以不用+，因为双引号包裹的字符串，不会去除空格，也不用URL编码直接会带入查询
