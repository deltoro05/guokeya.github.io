Pwnhub easythink(TP5sql注入)

Posted on 2020-07-05，2 min read

进入就给了源码。Thinkphp

 <?php
namespace app\index\controller;

use app\index\model;
use think\Container;

class Index
{
    public function index(){
        highlight_file(__FILE__);
        $model = new model\Index();
        
        $id = @$_GET['id'];

        $model->where(["id"=>$id])->select();
    }
}

很明显的Thinkphp sql注入。
打个单引号报错。得知版本是V5.1.37 TLS
并且知道了后台语句是

SELECT * FROM `index` WHERE `id` = $_GET['id']

但是这里没有回显。嘚通过布尔盲注或者时间盲注来整。

通过报错。我们得知flag表中有flag字段。
百度下。知道Thinkphp有个报错注入

然而。不能带select。如果有select子句。就会变成SQLSTATE[HY000]: General error: 1105 Only constant XPATH queries are supported

那么。我们可以通过判断语句返回值。然后如果是1。就报错。
